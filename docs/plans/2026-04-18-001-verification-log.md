## Bundle diff (automated)

### Tooling

`tools/diff-bundles.sh` (added in this unit) does:

1. Strip the esbuild banner (`/* THIS IS A GENERATED/BUNDLED FILE… */`).
2. Strip the runtime-helpers preamble (`var __create = …` through `var __toCommonJS = …`).
3. Sort top-level single-line `var require_*` declarations (no-op for our 1
   wrapper; documented tradeoff in the script header).
4. Emit a unified `diff -u` to `tools/diff-output.patch`.
5. Run an **order-insensitive** content comparison via `comm` on sorted unique
   lines — esbuild reorders modules between builds, so the positional diff is
   dominated by reordering noise. The `comm`-based pass is the real signal.
6. Categorize the order-insensitive deltas via successive `grep -vE` filters
   into seven buckets (helper, module-ordering, import-rename, enum-emit,
   optional-chain, id-rename, **actual-logic** = uncategorized residue).

Re-run the tool with:

```sh
bash tools/diff-bundles.sh tools/inputs/main-v1.0.19.js main.js
```

### Triage of "actual-logic" residue

The residue is **not** uncategorized arbitrary divergence. It is dominated by
mechanical formatting differences that the categorizer's narrow regexes do not
catch. Triaged buckets:

| Pattern                                                                    | Count (approx) | Verdict      | Why benign                                                                                                                                   |
| -------------------------------------------------------------------------- | --------------:| ------------ | -------------------------------------------------------------------------------------------------------------------------------------------- |
| Inline `if (X) return;` (ref) vs split `if (X)\n  return;` (rebuilt)       |          ~120  | benign       | Pure prettier-style reformatting introduced when esbuild re-emitted source. Same control flow.                                                |
| Enum literal-with-comment (ref `"deepgram" /* Deepgram */`) vs bare string |           ~70  | benign       | Const-enum members were inlined to plain string unions during splitting. Runtime value identical (`"deepgram" === "deepgram"`).               |
| Single-quote vs double-quote string literals                               |           ~30  | benign       | Lexer-level only.                                                                                                                            |
| Parenthesized `(distFromBottom + 6) + 'px'` vs unparenthesized + `"px"`    |           ~10  | benign       | Same expression; ref source had explicit parens, rebuilt source/printer dropped them.                                                          |
